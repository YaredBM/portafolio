Microsoft Azure Diseño Web con Wordpress Analítica Web

Microsoft Azure es una plataforma en la nube que ofrece una amplia gama de servicios y herramientas para el desarrollo, la implementación y la administración de aplicaciones y servicios en la nube. Es una solución completa y escalable que me permite alojar mis aplicaciones y datos en la nube, lo que significa que puedo acceder a ellos desde cualquier lugar y en cualquier momento.

WordPress es una plataforma de gestión de contenido para sitios web que permite crear y administrar un sitio web sin tener conocimientos técnicos profundos en programación. Con WordPress, es fácil crear y publicar contenido en una variedad de formatos, como artículos de blog, páginas web, galerías de fotos y mucho más.

La analítica web es una herramienta fundamental para entender cómo funcionan los sitios web. Se trata de un conjunto de técnicas y herramientas que permiten analizar y medir el tráfico de visitas en un sitio web, así como el comportamiento de los usuarios dentro del mismo. Se pueden obtener datos precisos sobre el número de visitas.

Proyectos
Asistente Virtual Sistema de Facturación Granja Hidropónica
Automatizada		 Detector de salud en plantas
con Custom Vision AI

Diseño de una asistente virtual programada con Python orientada a las matemáticas, que es capaz de responder a través de comandos de voz a las preguntas que se presentan más frecuentemente en la materia. Además, permite una interacción óptima con el usuario a través de juegos interactivos.

Sistema de facturación en Excel a partitr de programación en Python usando distintas librerías, primordialmente Tkinter y Openpyxl que es el que hace posible la creación de la ventana sobre la cual se va a trabajar, y el sistema login y signup correspondiente, en sí, fueron estas las librerías que hicieron este proyecto posible.

Construcción de una granja hidropónica que permita el buen desarrollo de la vida vegetal y animal junto al componente tecnológico con programación en Arduino para regular los procesos biológicos y químicos manteniendo el pequeño ecosistema controlado y estable.

Detección de salud de plantas con inteligencia artificial, utilizando Custom Vision para ello, entrenamiento de varias imágenes y etiquetas y nos da probabilidades de cada uno de los aspectos vitales que se deben tener en cuenta para el cuidado de una planta.
